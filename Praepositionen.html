Akkusativ Always Accusative

These always take Akkusativ — no exceptions. Mnemonic: FUDGE BOP (für, um, durch, gegen, entlang, bis, ohne, pro)

Dativ Always Dative

These always take Dativ. Mnemonic: ABS GANZ MV (aus, bei, seit, gegenüber, ab, nach, zu, mit, von)

Wechsel Two-Way (Akkusativ or Dativ)

Wohin? (motion toward) → Akkusativ · Wo? (location, no motion) → Dativ. These 9 are the ones you really need to internalize.

Genitiv Genitive

More formal / written. In spoken Swiss German many of these get replaced with Dativ + von, but you'll see them in writing and business contexts.

The Wechselpräpositionen Rule
Wo? (where — stationary) → Dativ
Das Buch liegt auf dem Tisch. (It's sitting there.)

Wohin? (where to — movement) → Akkusativ
Ich lege das Buch auf den Tisch. (I'm placing it there.)

The preposition stays the same — only the article changes. If the thing ends up somewhere new → Akkusativ. If it's already there → Dativ.
Contractions (common in speech & writing)
an + dem = am · an + das = ans · auf + das = aufs · bei + dem = beim · in + dem = im · in + das = ins · von + dem = vom · zu + dem = zum · zu + der = zur
